- **shadcn v4 style naming change:** The plan specified "New York" style, but shadcn v4 replaced the style system with presets. `base-nova` is the direct successor to New York — same Neutral base color, CSS variables enabled, Radix UI primitives. - **vitest --passWithNoTests:** Vitest exits code 1 when no test files exist. Added `--passWithNoTests` flag to `npm test` script so it exits 0 cleanly before test files are created in Plan 02.
- **node_modules cleanup:** The initial git commit accidentally included node_modules (no .gitignore existed). Created .gitignore and unstaged all node_modules files in Task 1 commit to keep repo clean.
