Run the following commands in sequence:

1. Scaffold Next.js:
```bash
npx create-next-app@latest . --typescript --tailwind --eslint --app --src-dir --import-alias "@/*"
```
Accept defaults when prompted. This creates the full project structure with `src/` directory.

2. Initialize shadcn/ui:
```bash
npx shadcn@latest init -t next
```
When prompted, select:
- Style: New York
- Base color: Neutral
- CSS variables: Yes
Accept the default alias detection (reads tsconfig.json, pre-fills `@/*` -> `./src/*`).

3. Install next-themes:
```bash
npm install next-themes
```

4. Create `src/components/theme-provider.tsx` with 'use client' directive:
```tsx
'use client'

import * as React from 'react'
import { ThemeProvider as NextThemesProvider } from 'next-themes'

export function ThemeProvider({
  children,
  ...props
}: React.ComponentProps<typeof NextThemesProvider>) {
  return <NextThemesProvider {...props}>{children}</NextThemesProvider>
}
```

5. Update `src/app/layout.tsx` to wrap children with ThemeProvider:
- Import `ThemeProvider` from `@/components/theme-provider`
- Add `suppressHydrationWarning` to the `<html>` tag
- Wrap `{children}` inside `<body>` with:
```tsx
<ThemeProvider
  attribute="class"
  defaultTheme="system"
  enableSystem
  disableTransitionOnChange
>
  {children}
</ThemeProvider>
```

6. Verify TypeScript strict mode is enabled in `tsconfig.json`. The `create-next-app` command sets `"strict": true` by default. Confirm it is present.

7. Verify `src/lib/utils.ts` exists (created by shadcn init) and exports the `cn()` function using `clsx` + `tailwind-merge`.
